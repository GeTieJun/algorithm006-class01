#二分查找 理解
***
####使用二分查找，寻找一个半有序数组 [4, 5, 6, 7, 0, 1, 2] 中间无序的地方 理解
***
* 二分查找法条件：
    * 有界性（存在上下边界）
    * 有序性（升序|降序）
    * 通过索引可以获取到

* 针对寻找一半有序数组，例如[4, 5, 6, 7, 0, 1, 2],重要条件在于中间值
```nums[mid]```的比较，以及```left``` 和 ```right``` 脚标，具体代码如下(数组默认升序)：
```
    if(nums[mid] > nums[mid + 1]){
        return nums[mid + 1]; 
    }

    if(nums[mid - 1] > nums[mid]){
        return nums[mid];
    }

    if(nums[mid] > nums[0]){
        left++;
    }else{
        right--;
    }
```
